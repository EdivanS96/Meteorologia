# Meteorologia

Este repositório está em desenvolvimento e será o espaço onde vou armazenar e organizar scripts, análises e ferramentas que desenvolver ao longo da minha graduação, com foco em meteorologia e áreas relacionadas, como climatologia, sensoriamento remoto e análise ambiental.

---

## Sobre o projeto

Ainda não há conteúdo, mas a ideia é incluir progressivamente códigos para o processamento e análise de dados meteorológicos e ambientais, incluindo:

- Análise de imagens de satélite (Landsat, Sentinel, etc.)  
- Cálculo de índices ambientais (NDVI, LST, Albedo, etc.)  
- Processamento de séries temporais meteorológicas.
- Ferramentas para visualização e exportação de dados. 

---

## Tecnologias e ferramentas

Pretendo utilizar, principalmente:

- Google Earth Engine (GEE) — processamento remoto de imagens de satélite  
- Python / JavaScript — linguagens para desenvolvimento dos scripts  
- Bibliotecas específicas para análise de dados e geoprocessamento  

---

## Como usar

Os scripts ainda estão sendo desenvolvidos, mas, futuramente, poderão ser executados diretamente no Google Earth Engine (para códigos em JS) ou localmente (para scripts em Python). É recomendável ter conhecimentos básicos em geoprocessamento e análise ambiental para melhor aproveitamento.
